# Create New Module

```bash
php artisan app:make-module
```

It will ask for the module name and create a new module with all the necessary files and folders. Including the following:
- Controller
- Model
- Actions
- Requests
- Views (inertia JSX)
- Routes
- Migrations
- Seeders
- Factories
